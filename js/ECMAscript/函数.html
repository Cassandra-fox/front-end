函数

命名规则：

  • 函数名由字母、数字、下划线和美元符号组成，且不能以数字开头
  • 函数名不能是JavaScript的关键字或保留字
  • 函数名区分大小写
  • 函数名应具有描述性，便于理解其功能，常用动词

函数的参数：

  • 参数是函数定义时指定的变量，用于接收调用函数时传递的值
  • 参数可以有默认值，如果调用函数时没有传递对应的参数，则使用默认值
  • 参数可以是任意类型的数据，包括基本类型和引用类型
  • 函数可以有多个参数，参数之间用逗号分隔
  • 函数可以没有参数

函数的返回值:
return后面的语句不会被执行，会立刻结束当前函数的执行
多个返回值可以用数组或对象的形式返回
如果函数没有返回值，默认返回undefined

补充：
相同名称的函数会覆盖前面的函数
函数可以作为参数传递给另一个函数
实参和形参的个数可以不一致，未传递的实参对应的形参值为undefined
参数不匹配

作用域：
函数内部定义的变量只能在函数内部访问，函数外部无法访问
函数可以访问外部作用域的变量
函数可以嵌套定义，内层函数可以访问外层函数的变量
全局变量可以在任何地方访问，但不建议滥用全局变量
特殊情况：函数内部如果没有使用var、let、const定义变量，则该变量会成为全局变量
严格模式下，函数内部未声明的变量不能成为全局变量
优先级：局部变量 > 外层函数变量 > 全局变量

匿名函数：
没有函数名的函数称为匿名函数
匿名函数通常用于一次性使用的场景，如事件处理、定时器等
匿名函数可以赋值给变量，从而通过变量名调用该函数
函数表达式：将函数赋值给变量的方式称为函数表达式
和具名函数的差别：
1.函数表达式不会被提升，必须在定义后才能调用
2.具名函数可以在任何地方调用

立即执行函数（IIFE）：
立即执行函数是一种特殊的匿名函数，定义后立即执行
(function () { console.log("立即执行函数") })()
立即执行函数通常用于创建独立的作用域，避免变量污染全局作用域
立即执行函数可以传递参数
多个立即执行函数之间要加分号隔开，防止语法错误

逻辑中断： 短路运算符：&&和||
逻辑与（&&）：如果第一个操作数为真，则返回第二个操作数，否则返回第一个操作数
逻辑或（||）：如果第一个操作数为假，则返回第二个操作数，否则返回第一个操作数

转换为布尔型

以下值转换为false：
false
0
-0
0n（BigInt类型的零）
""（空字符串）
null
undefined
NaN
其他值转换为true
