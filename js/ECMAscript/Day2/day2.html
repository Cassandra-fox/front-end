赋值运算符

= += -= *= /= %=

一元运算符

++ -- + - ~ !

比较运算符

==(左右两边值是否相等) ===(值和类型都相等) != !==(值不相等或类型不相等) < <= > >= << >> >>>

比较运算符有隐式转换 == 会把字符串转换成数字再比较
字符串比较，是在比较对应的ASCII码
涉及到NaN的比较，结果都是false
比较小数时，可能会出现精度问题。先转化成整数再比较

逻辑运算符

&&(与) ||(或) !(非)

运算符优先级

一元运算符 > 算术运算符 > 比较运算符 > 相等运算符 > 逻辑运算符 > 赋值运算符 逻辑与的优先级高于逻辑或

语句

分支语句

if (条件) { // 代码块 } else { // 代码块 }

if (条件1) { // 代码块 } else if (条件2) { // 代码块 } else if (条件3) { // 代码块 } else { // 代码块 }

switch (表达式) { case 值1: // 代码块 break; case 值2: // 代码块 break; default: // 代码块 break; }

//三元运算符 条件 ? 表达式1 : 表达式2

循环语句

for (初始化; 条件; 迭代) { // 代码块 }
用来打印数组

let i = 0; while (条件) { // 代码块 i++; }

当明确循环次数时，优先使用for循环
当循环次数不明确时，优先使用while循环

循环退出

break: 退出整个循环
while (true) 构建无限循环需要用到break
例子：重女来袭
continue: 退出本次循环，进入下一次循环

数组

增删改查

let arr = [元素1, 元素2, 元素3]
arr[索引] = 新元素
arr.push(新元素) //在末尾添加，并返回数组的长度
arr.pop() //删除末尾元素
arr.unshift(新元素) //在开头添加
arr.shift() //删除开头元素
arr.splice(起始索引, 删除个数, 新元素1, 新元素2) //从指定位置添加或删除元素
arr.length //数组长度
arr.indexOf(元素) //查找元素的索引，找不到返回-1
arr.includes(元素) //查找数组是否包含某个元素，返回true或false
