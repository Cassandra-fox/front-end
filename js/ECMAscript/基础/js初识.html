更改制表符

是什么

运行在客户端的编程语言，实现人机交互。 网页特效(监听用户行为做出反应) 表单验证 数据交互(获取后台的数据渲染到前端) 服务端编程（node.js）

组成

ECMAscript (语法规范:变量，分支语句，循环语句，对象等) Web APIs: DOM(页面文档对象模型) BOM(浏览器对象模型) DOM:操作文档，对页面元素进行移动大小，添加操作等操作 BOM:操作浏览器，页面弹窗，检测窗口宽度，存储数据到浏览器等

注释写法： 单行注释：// 这是单行注释 ctrl+/ 多行注释：/* 这是多行注释 */ ctrl+shift+A

结束符 分号; 语句结束符，可以省略。风格统一。还是不写吧。

输入输出

用户向计算机输入信息，计算机向用户输出信息

输出 alert() 弹窗 document.write() 向文档写入内容 console.log() 控制台输出,程序员调试使用

输入 prompt() 弹窗输入 confirm() 弹窗确认

js代码执行顺序: 按文档流顺序执行，alert和prompt会阻塞代码执行

字面量

literal 字面量 直接量，直接写在代码中的固定值 数字字面量 123 3.14 -5 字符串字面量 'abc' "abc" `abc` 布尔字面量 true false 数组字面量 [1,2,3] 对象字面量 {name:'张三',age:20}

变量

变量：存储数据的容器 声明变量：let 变量名; const 变量名=值; 命名规则：字母，数字，下划线，$符号，不能以数字开头，不能是关键字 字母严格区分大小写。小驼峰命名法：myNameIs 赋值：变量=值;

var现在已经不推荐使用了 可以先使用再声明，不合理 声明过的变量可以重复声明，不合理 没有块级作用域

数组

存储一组数据 let arr = [1,2,3,'abc',true]; 通过索引访问数组元素，索引从0开始 arr[0] 访问第一个元素 arr.length 获取数组长度

常量

常量：一旦声明就不能修改的变量，不能重新赋值，声明时必须赋值 声明常量：const 常量名=值; 命名规则同变量

数据类型

js是弱数据类型语言，变量不需要指定数据类型，数据类型会自动转换 数据类型分为两大类：基本数据类型和引用数据类型 基本数据类型：Number数字，String字符串，Boolean布尔，Undefined未定义，Null空，BigInt大整数，Symbol符号 引用数据类型：Object对象，Array数组，Function函数 typeof 运算符可以检测数据类型 NaN 非数字,代表计算错误，是一个不正确的计算结果

算术运算符：+ - * / %取模 ++ -- 赋值运算符：= += -= *= /=

类型转换

prompt输入的值都是字符串类型 隐式转换：自动进行类型转换 字符串和数字进行+运算会进行字符串拼接 其他运算符会将字符串转换为数字进行计算 布尔值true转换为1，false转换为0 +作为正号解析可以转换为数字 显式转换：手动进行类型转换 字符串转数字：Number() parseInt()取整 parseFloat() 会把不是数字的部分去掉 数字转字符串：String() toString()
