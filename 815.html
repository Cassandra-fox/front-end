1.表格标签

表格：用来展示数据

基本语法：

表头一 表头二 表头三
单元格内的文字 第二个单元格 第三个单元格
第二行 第二个单元格 第三个单元格

2.表格属性

一般通过css
align (left,center,right) 表格相对周边元素的对齐方式
border (1/无) 规定表格是否有边框，默认为""，表示没有边框
cellpadding 规定单元边沿与其内容之间的空白，默认为1像素
cellspacing 规定单元格之间的空白，默认2像素
width 规定表格的宽度
表头一 表头二 表头三
单元格内的文字 第二个单元格 第三个单元格
第二行 第二个单元格 第三个单元格

2.表单控件

  1. input输入表单元素
  2. select下拉表单元素
  3. textarea文本域元素

1.input输入表单元素

用于收集用户信息
包含一个type属性，根据不同的属性值，输入字段拥有很多种形式，（文本框、复选框、掩码后的文本控件、单选按钮、按钮 等）
input标签为单标签

name属性定义input元素的名字
value属性规定input元素的值，在文本框中可显示
checked属性值checked，规定此input元素首次加载时应被选中
maxlength属性规定输入字段中的字符的最大长度

type属性来指定不同的控件类型:
text 单行的输入字段，用户可在其中输入文本。默认宽度为20个字符
password 定义密码字段，该字段中的字符被掩码
radio 单选按钮
checkbox 定义复选框
reset 重置
submit 提交
button 定义可点击按钮,一般通过js
file 上传文件

2.label标签

为input元素定义标注(标签)
用于绑定一个表单元素,当点击label标签内的文本时,浏览器会自动选择对应的表单元素,增加用户体验
for对应id
